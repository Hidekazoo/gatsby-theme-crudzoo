The template for book reviews can display a score and set a link in the title section. The score can be set from 1 to 5. Other styles are the same as for normal articles.

## Usage

```
---
id: "article-1"
title: Book Review Template
image: "./image1.jpg"
date: "2020-12-30T16:51:45+09:00"
tags: ["review"]
spoiler: "Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Gravida neque convallis a cras."
template: "book-review"
score: "5"
link: "https://example.com"
---
```
